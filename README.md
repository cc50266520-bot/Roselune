# Rosélune – AI-Based Makeup Recommendation System

Rosélune is an online makeup store that uses AI-based skin tone detection to recommend suitable makeup products such as foundation, lipstick, and blush.

## Features
- User registration & login
- Upload photo or use camera
- Automatic skin tone detection (light / medium / dark)
- Personalized makeup recommendations
- Shopping cart & checkout
- Admin dashboard (manage products & orders)

## Technologies Used
- PHP
- MySQL (phpMyAdmin / SQL Server for design)
- HTML, CSS, JavaScript
- XAMPP
- face-api.js

## How to Run Locally
1. Install XAMPP
2. Copy project to:
   C:\xampp\htdocs\HasnaaChakik_51831003
3. Start Apache & MySQL
4. Create database: `roselune`
5. Import SQL file (if provided)
6. Open:
   http://localhost/HasnaaChakik_51831003/index.php

## Admin Access
Set `is_admin = 1` in the users table for admin access.
